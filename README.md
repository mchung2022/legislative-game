# 🇹🇼 台灣選舉制度教育網頁遊戲——「民主選戰大進擊：台灣選舉制度模擬挑戰賽！」

這是一款專為**國中與高中公民課程**設計的一頁式網頁互動遊戲，以中華民國（台灣）的選舉制度為主題。透過精緻的視覺設計、動態模擬與小遊戲，讓學生在遊玩中深刻理解台灣的核心參政權利與選務機制，增進公民素養。

---

## 🎮 遊戲特色與關卡流程

遊戲共有四大核心關卡，完整涵蓋台灣的各項代表性選舉與罷免複決機制：

1. **關卡一：選民參政權與公民投票門檻**
   - **任務 1：選民資格審查**：將不同年齡與身分的卡片拖曳/點選分發到正確的權利區（18歲僅公投、20歲具一般選舉權、23歲可參選立委、40歲可參選總統）。
   - **任務 2：公投「雙重門檻」模擬器**：透過可調式滑桿與動態柱狀圖，模擬「是否同意高中生延後上學」公投案。體驗《公民投票法》規定的雙重門檻（① 同意票大於不同意票、② 同意票達總投票權人四分之一以上），直觀理解為何有時公投「同意票較多卻依然不通過」。

2. **關卡二：總統大選決戰（相對多數決）**
   - **選戰造勢與策略部署**：扮演候選人「民主希望隊」的選戰總幹事，在青年居住、銀髮福利與科技綠能中選擇核心造勢政見。
   - **即時開票看板**：啟動大選投開票，以動態計票條與計票音效呈現三方爭霸。帶出台灣總統大選採用的「單一選區、相對多數決」（贏者全拿）制度特徵，思辨其優缺點。

3. **關卡三：立委選舉大鬥智（單一選區兩票制）**
   - **神聖兩票投開票體驗**：模擬投下「區域立委票（選人）」與「政黨票（選黨）」，體驗選票投入票箱的物理滾動動畫。
   - **政黨票席次計算機**：玩家可手動調整 6 個政黨的政黨得票百分比，並即時啟動計算。系統將依照實際的 **5% 得票率門檻** 篩選政黨，並以**最大餘數法**分配 34 席不分區立委，套用**女性保障名額（不得低於 1/2）**輪替排序规则。
   - **議場席次分佈圖**：使用 HTML/SVG 動態繪製立法院議場席次半圓，即時點亮各黨當選席次。

4. **關卡四：九合一地方大選（選票分發挑戰）**
   - **投票所選務模擬**：在限時 30 秒內扮演選務人員。面對前來領票的選民身分證件（如一般選民、原住民、戶籍在新竹縣、台北市、或台中市和平區山地原住民區），正確分發選票組合（如直轄市 3 張、一般縣市 5 張、原住民區 5 張等）。挑戰你的快速反應力與制度熟悉度！

5. **公民結算與民主證書**
   - 彙整四大關卡的得分，自動生成專屬的「台灣民主素養合格公民證書」，蓋有「中選印」風格的虛擬朱印。
   - 支援**列印或另存 PDF**，方便學生作為公民課的課堂學習單提交。

## 🛠️ 技術架構

本專案本著**輕量、流暢、免安裝、100% 離線可用**的原則開發：
- **HTML5 & CSS3**：響應式玻璃擬物化 (Glassmorphism) 深藍色漸層科技感介面，自適應支援電腦、手機與平板，方便在教室人手一台平板下遊玩。
- **Vanilla ES6+ JS**：不依賴任何外部前端框架（如 React、Vue 等），極速載入且完全無跨域安全限制作業。
- **Web Audio API**：利用瀏覽器音頻引擎，**即時以代碼合成遊戲音效**（包含按鈕啵啵聲、蓋章衝擊聲、投票箱咚咚聲、答對琶音、答錯低鳴與勝利歡呼），不需要載入任何外部 `.mp3` 或 `.wav` 音檔，避免路徑錯誤或跨域載入阻擋。
- **SVG / Canvas**：以 SVG 渲染精緻的半圓形議場席次分佈，以 Canvas 繪製關卡通過與結算時的五彩紙屑（Confetti）物理重力下落特效。
