# 國中公民課教育遊戲——「法案奇幻冒險：三讀闖關大作戰！」

這是一款專為**國中生**設計的網頁互動遊戲，將中華民國立法院的法律制定程序（俗稱「立法三讀」）包裝成趣味十足的冒險闖關遊戲。透過角色扮演、趣味迷你遊戲、公民知識問答與生動的敲槌/表決動畫，讓枯燥的政治與法律課程變得活潑有趣，加深學生的學習記憶。

---

## 🎮 遊戲特色與關卡流程

遊戲共有五大核心關卡，完整還原現實中的立法流程：

1. **提案與一讀（連署收集小遊戲）**
   - 學生需在限時內點擊飄浮的立委信件，收集至少 **15 位立委的連署簽名**以成功提案。
   - 點擊發光的議事槌，模擬一讀朗讀案由後送交委員會審查。
2. **委員會審查與協商（正反觀點分類 + 朝野黨團協商）**
   - 拖曳或點擊卡片，將公聽會的正反意見分類至對應框。
   - 朝野意見分歧時，操作三個政黨的讓步滑桿，讓藍、綠、黃三方的指針重疊在「共識發光區」以完成黨團協商。
3. **決戰二讀會（公民知識拉票 + 電子表決模擬）**
   - 逐條審議的關鍵！回答 3 題關於立法院、讀會制度、覆議程序的公民選擇題。
   - 答對越多題，能拉攏越多游離票立委（表決板上點亮綠燈）。
   - 啟動電子表決器，動態開票看贊成票是否能突破過半數的 **57 票**門檻。
4. **三讀會把關（文字校對 + 終極敲槌儀式）**
   - 三讀原則上只能進行「文字修正」。找出法案草案中的 2 個錯字或不當用詞，點擊修正。
   - 校對完成後，玩家可親自敲擊院長議事槌，伴隨震撼音效宣告法案三讀通過！
5. **行政院覆議挑戰（快速點擊維持原案）**
   - 行政院長認為法案窒礙難行，移請立法院覆議！
   - 玩家必須在 8 秒內快速狂點「維持原案」按鈕，凝聚全體委員意志。達 57 票以上即成功駁回覆議！
6. **總統公布與頒發證書**
   - 總統依法公布，法案正式成為法律！
   - 遊戲生成精美的「法律公布證書」，印有玩家的大名與所推動的法案，提供「螢幕截圖」功能。

## 🛠️ 技術架構

本專案本著**輕量、流暢、免安裝、100% 離線可用**的原則開發：
- **HTML5 & CSS3**：建立響應式玻璃擬物化 (Glassmorphism) 介面，支援手機與平板（方便課堂人手一台平板操作）。
- **Vanilla ES6+ JS**：無外部框架依賴，極速載入。
- **Web Audio API**：利用瀏覽器音頻引擎**即時合成遊戲音效**（包含清脆按鍵聲、啵啵泡泡聲、木槌敲擊聲、答對升調琶音、答錯低沉音、勝利歡呼噪聲），不需要加載任何外部 MP3 檔案，徹底避免因瀏覽器 CORS 政策或檔案路徑錯誤導致音效失效。
- **Canvas**：用於三讀通過與證書結算時的五彩紙屑 (Confetti) 物理模擬特效。

## 🚀 本地執行方式

本專案無需任何建置工具，雙擊即可執行：
1. 下載專案資料夾後，找到 `index.html`。
2. 在您的瀏覽器（建議使用 Chrome、Edge 或 Safari）中**按兩下開啟 `index.html`**，即可開始遊玩！

## 🌐 部署至 GitHub Pages

為了讓學生與老師能直接用手機或平板連線上網遊玩，您可以將此專案部署到 GitHub Pages（完全免費）：

### 步驟 1：建立 GitHub 倉庫 (Repository)
1. 登入您的 [GitHub](https://github.com/) 帳號。
2. 點擊右上角 **New** 新增一個儲存庫。
3. 儲存庫名稱（Repository name）可以輸入例如：`legislative-game`。
4. 選擇 **Public**（公開），然後點擊 **Create repository**。

### 步驟 2：上傳專案程式碼
若您熟悉 Git 指令，可以在本專案資料夾路徑下執行以下指令：
```bash
git init
git add .
git commit -m "Initial commit: Legislative three readings game"
git branch -M main
git remote add origin https://github.com/您的帳號名稱/legislative-game.git
git push -u origin main
```
*如果不熟悉指令，也可以直接在 GitHub 網頁上選擇「uploading an existing file」，將 `index.html`、`style.css`、`app.js` 與 `README.md` 拖曳上傳並 Commit 提交。*

### 步驟 3：啟用 GitHub Pages
1. 在您的 GitHub 儲存庫頁面，點選上方的 **Settings**（設定）。
2. 在左側選單中找到並點選 **Pages**。
3. 在 **Build and deployment** 下的 **Source** 選擇 `Deploy from a branch`。
4. 在 **Branch** 下拉選單中選擇 `main` 分支與 `/ (root)` 資料夾，並點選右側的 **Save**（儲存）。
5. 等待約 1 到 2 分鐘，重新整理頁面，您就會在上方看到您的專案專屬網址（例如：`https://您的帳號名稱.github.io/legislative-game/`）。複製該網址，就可以分享給國中生和老師們使用了！
